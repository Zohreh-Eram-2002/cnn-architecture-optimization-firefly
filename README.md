# cnn-architecture-optimization-firefly
# CNN Architecture Optimization using Firefly Algorithm

## Overview

This project applies a **metaheuristic optimization algorithm (Firefly Algorithm)** to automatically design and optimize Convolutional Neural Network (CNN) architectures.

The model is evaluated on the CIFAR-10 dataset.

---

## Objective

* Automatically search for optimal CNN architectures
* Combine Deep Learning with Bio-inspired Optimization
* Improve model performance without manual tuning

---

## Key Concepts

* Convolutional Neural Networks (CNN)
* Firefly Algorithm (Swarm Intelligence)
* Neural Architecture Search (NAS)

---

## Dataset

* CIFAR-10 (automatically downloaded via PyTorch)

---

## How It Works

1. Initialize random CNN architectures
2. Train each model briefly
3. Evaluate accuracy
4. Move weaker solutions toward stronger ones
5. Repeat for multiple iterations

---

## Output

* Best CNN architecture found
* Corresponding accuracy

---
